Centro de Ayuda Admin

Bienvenido al ecosistema administrativo de D'Antojitos©. Aquí encontrarás las directrices para operar la plataforma con máxima eficiencia y fluidez.

Gestión Fluida

La interfaz ha sido optimizada para ofrecer una respuesta inmediata. Cada cambio realizado impacta directamente en la experiencia de tus clientes sin tiempos de espera.

Seguridad

El sistema cuenta con capas de verificación de rol. Como administrador, tus acciones son definitivas para el inventario y la contabilidad del negocio.

Gestión de Usuarios

Jerarquía Administrativa

El control de acceso es fundamental para la seguridad operativa:

  • Asignación de Roles: Capacidad de elevar cuentas a nivel Administrativo.
  • Límite de Seguridad: Se recomienda un máximo de 3 administradores activos.
  • Privilegios: Acceso exclusivo a reportes financieros y configuración de pagos.

Gestión de Productos

Control de Inventario y Catálogo

Centraliza la oferta comercial de tu tienda con herramientas de edición dinámica:

  • Gestión Integral: Creación, edición y retiro de productos en segundos.
  • Sincronización: El catálogo público refleja cambios instantáneamente.
  • Stock Inteligente: Reposición automática en caso de cancelaciones o limpiezas de carrito.

Gestión de Pedidos

Logística y Seguimiento

Administra el flujo de ventas con herramientas de filtrado avanzado:

  • Filtros Avanzados: Localiza pedidos por cédula, nombre o estado de factura.
  • Reportes: Generación de cierres de caja en formato PDF por mes y año.
  • Priorización: Sistema de "fijado" para destacar pedidos urgentes al inicio.
  • Control de Pagos: Switch de validación para saldos pendientes y entregas.

Estrategia de Publicidad

Marketing Visual

Configura la identidad visual y promocional de la plataforma:

  • Alertas Globales: Notificaciones con imagen para campañas flash.
  • Carrusel Dinámico: Gestión de banners principales con vista previa.
  • Persistencia: Recuerda usar el botón "Guardar todos los cambios" para publicar.

Configuración de Pagos

Canales de Recaudo

Administra los métodos de pago disponibles para tus clientes:

  • Billeteras: Vinculación de Nequi, Daviplata y cuentas bancarias.
  • Factura PDF: Sincronización automática de datos en el comprobante del cliente.
  • Previsualizador: Verifica la estética de los datos de pago antes de confirmar.
